# Rainbow Library Suggestions Board

Noticed a bug, have a great new feature in mind? 
Make an issue in this repository, and we'll see what we can add!

We do ask that you follow the issue templates, to ensure that we have
a clear understanding of what you're suggesting, and how we can address
it.

## New to GitHub Issues?
Take a peek at GitHub's documentarion here:
https://docs.github.com/en/issues/tracking-your-work-with-issues/learning-about-issues/quickstart

## Technical concerns only!
If your concern has to do with something non-technical, please make a ticket in our Discord server.

## Notice to volunteers:
Keep in mind that everything here is public. If you don't know if something 
can be discussed here, consult with your Lead.
